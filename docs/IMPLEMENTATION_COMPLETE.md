## üîß How It Works

### Mock Data Flow
```
User Action ‚Üí useFakeEdit Hook ‚Üí sessionStorage ‚Üí Merge with Mock Data ‚Üí Display
```

### Search Flow
```
User Types ‚Üí Debounce 300ms ‚Üí mockSearchService ‚Üí Calculate Relevance ‚Üí Display Top 10
```

### AI Agent Flow
```
User Question ‚Üí geminiService ‚Üí Check API Key ‚Üí Gemini API OR Mock Response ‚Üí Chat UI
```

### Admin Stats Flow
```
Load Dashboard ‚Üí mockAdminService ‚Üí Calculate from mockLeads/Projects ‚Üí Display with Supabase Context
```

    applyFakeEdit,
    mergeWithFakeEdits,
    resetAllEdits,
    fakeDelete,
    isDeleted
  } = useFakeEdit();

  const handleEdit = (projectId: string, changes: any) => {
    applyFakeEdit('project', projectId, changes);
  };

  const handleDelete = (projectId: string) => {
    fakeDelete('project', projectId);
  };

  const displayProjects = mockProjects
    .map(p => mergeWithFakeEdits('project', p))
    .filter(p => !isDeleted('project', p.id));
